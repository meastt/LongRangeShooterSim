# LongRangeShooterSim

Repository reset to a clean slate. Add Unity, Godot, shared code, and docs as you rebuild.

## Local hygiene

If you still have old Unity or Godot cache folders on disk from before the reset, you can remove them manually (they are not tracked):

- `Library/`, `Logs/`, `UserSettings/` (Unity)
- `godot/.godot/` (Godot import cache)
